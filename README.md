# Invoice Billing Manager

A full-stack SaaS application for managing customers, invoices, subscriptions, and payments.

## Tech Stack

### Frontend
- React
- Vite
- React Router
- Axios

### Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL

### Features
- JWT Authentication
- Customer Management
- Invoice Generation
- PDF Download
- Stripe Subscription Billing
- Multi-Tenant SaaS
